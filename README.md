# Many-body localisation — exact diagonalisation

Using exact diagonalisation of the random-field Heisenberg spin-1/2 chain:

$$H = \sum_{i=1}^{L} \left[ h_i \hat{S}^z_i + J \hat{\mathbf{S}}_i \cdot \hat{\mathbf{S}}_{i+1} \right]$$

with periodic boundary conditions, $J = 1$, and $h_i \in [-W, W]$ uniform random.

## Dependencies

| Library | Purpose |
|---------|---------|
| Eigen 3 | sparse matrix construction |
| LAPACK / OpenBLAS | dense diagonalisation (`dsyev`) |
| LAPACKE | C interface to LAPACK |
| Python ≥ 3.10 | analysis & plotting |
| NumPy, Matplotlib | figures |

Install on Ubuntu/Debian:
```bash
sudo apt install libeigen3-dev liblapack-dev libopenblas-dev liblapacke-dev
pip install numpy matplotlib
```

## Build

```bash
cmake -B build
cmake --build build -j$(nproc)
```

Binary: `build/solve_random_heisenberg`

## Usage

### Single run
```bash
./build/solve_random_heisenberg <L> <W> <duplicate_number>
# e.g.
./build/solve_random_heisenberg 12 3.6 1
```

Writes `results/eigenvalues_P12_W3600_dupli1.txt` and the corresponding eigenvectors file.

### Full parameter sweep
```bash
# Edit L_values, W_start/end/step, dup_start/end at the top of the script first
bash scripts/sweep.sh
```

## Parameters

|--------|----------|---------------|

| $W$ | disorder strength | 0.5 – 12 |
| duplicates | independent disorder realisations | 1 – 10 per $(L, W)$ |
